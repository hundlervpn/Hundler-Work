# Hundler Work

Биржа заказов по типу hh.ru — площадка для **заказчиков** и **исполнителей**.
Поиск заданий, отклики, свои заказы, чаты и профиль. Адаптив: мобильный макет + полноценная ПК-версия.

## Стек

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**

Дизайн собран с применением принципов [make-interfaces-feel-better](https://github.com/jakubkrehel/make-interfaces-feel-better): концентрические радиусы, тени вместо бордеров, stagger-анимации, tabular-nums, оптическое выравнивание, `scale(0.96)` на нажатие.

Фирменная палитра: **чёрный · белый · красный · фиолетовый** (красный — доминирующий акцент, фиолетовый — второстепенный).

## Вкладки

- **Найти работу** — лента заказов с поиском и категориями
- **Мои отклики** — отправленные заявки со статусами (принят / на рассмотрении / отклонён)
- **Мои заказы** — размещённые задания со статусами и числом откликов
- **Чаты** — переписка с заказчиками и исполнителями
- **Профиль** — аккаунт, баланс, статистика

## Адаптивность

- **Мобильный**: верхние табы + нижняя навигация
- **ПК (lg+)**: боковое меню (sidebar), широкий контент, сетка карточек в 2–3 колонки

## Локальный запуск

```bash
git clone https://github.com/hundlervpn/Hundler-Work.git
cd Hundler-Work
npm install
npm run dev
```

Открой http://localhost:3000

## Сборка

```bash
npm run build
npm run start
```
